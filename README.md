# MineWeb-install
...
## Usage
```sh
wget https://raw.githubusercontent.com/fightmaxime/mineweb-install/master/mineweb-install.sh
chmod +x mineweb-install.sh
./mineweb-install.sh
```
## Compatibilité

Le script supporte ces OS:

|        |   |
|--------|---|
| Debian 8 | ❌  |
| Debian 9 | ✅ |
| Ubuntu 16.04 | ❌  |
| Ubuntu 18.04 | ❌  |
| CentOS 7 | ❌  |
| Fedora 28 | ❌  |
## Features
* MySQL 5.7
* PHP 7.2
* phpMyAdmin
